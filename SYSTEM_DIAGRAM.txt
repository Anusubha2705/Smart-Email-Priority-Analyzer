                        PRIORITY CLASSIFICATION
═══════════════════════════════════════════════════════════════════════════════

HIGH PRIORITY (Red Badge 🔴)
───────────────────────────
Triggers/Keywords: URGENT, CRITICAL, ASAP, IMMEDIATE, EMERGENCY, 
                   DOWN, FAILED, BREACH, ESCALATE, NOW, DEADLINE TODAY

Examples:
  ✓ "URGENT: System Down - Production Issues"
  ✓ "Critical Security Alert - Immediate Action Required"
  ✓ "Emergency Meeting - All Hands in 30 Minutes"


MEDIUM PRIORITY (Yellow Badge 🟡)
─────────────────────────────────
Triggers/Keywords: IMPORTANT, REVIEW, FEEDBACK, MEETING TOMORROW,
                   APPROVAL, ACTION REQUIRED, DECISION NEEDED, SOON

Examples:
  ✓ "Quarterly Report Review - Due This Week"
  ✓ "Team Meeting Tomorrow at 2 PM"
  ✓ "Project Status Update - Your Input Needed"


LOW PRIORITY (Green Badge 🟢)
────────────────────────────
Triggers/Keywords: FYI, REMINDER, INFORMATION, UPDATE, AVAILABLE,
                   OPTIONAL, NICE TO HAVE, NEWSLETTER, BIRTHDAY

Examples:
  ✓ "Weekly Newsletter - Company Updates"
  ✓ "Office Supplies Available in Lobby"
  ✓ "Parking Reminder - Sunday Maintenance"
